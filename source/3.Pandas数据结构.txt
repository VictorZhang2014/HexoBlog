Pandas是Python Data Analysis Library或者基于Numpy的一种工具，一种数据分析包，
该工具是为了解决数据分析任务而创建的。Pandas纳入了大量库和一些标准的数据模型。

官网：http://pandas.pydata.org/
Github: https://github.com/pandas-dev/pandas


https://zhuanlan.zhihu.com/p/24309547

如果想安装Numpy, Matplotlib, Scipy, Pandas，等等科学使用工具，
最好是使用Anaconda，Anaconda的官网： https://docs.anaconda.com/anaconda/
有MacOS的Installer，也有命令行安装的

Series，一维数组，与Numpy中的array类似，但是series只能存储相同类型的元素
- 数组与标签
- 可以通过标签选取数据
- 定长的有序字典

Dataframe，二维的表格型数据结构
- 表格型数据结构
- 行索引、列索引

如果你的Mac提示 `ImportError: No module named pandas` ，则你需要 `pip install pandas`
